# **10R sleeve MFG 予実管理モニター(ver.3)**

## **工場の生産予実をaws cloudwatch ダッシュボードで表示するために不稼働項目を入力・管理する仕組み**
---
## **ver.2からの変更点**
- 3Gr GLより入力時間が確保できないので簡略化してほしいと要望を受け、不稼働項目入力を削除(1SPM半休、全体夕礼、16:40終了、17:40終了)
- 入力項目の削減、入力の省力化・簡素化

## **簡素化項目**
- 生産数、生産開始時刻の標準値をプリセットし、入力なしで予実表示が可能
- 全工程の生産数/生産時刻の入力を1ページにまとめ、登録ボタンも1つにまとめた
- 生産予定、終了予定時刻は工程ごとにまとめて表示
---

## **使用言語**
&emsp;&emsp;Lambda　：　Python 3.8.1
</br>
&emsp;&emsp;入出力ページ(web page)　：　HTML5/css/javascript　

## **入力項目**
- 生産機種
- 当日生産数
- 生産開始時刻

## **出力項目**
- 生産日
- 生産機種
- 生産予定数
- 生産開始時刻
- 生産終了予定時刻
- 途中時刻での予定実績数 

## **ページ構成**
- 全工程の生産数/生産時刻の入力を1ページにまとめ、登録ボタンも1つにまとめた
- 生産予定、終了予定時刻は工程ごとにまとめて表示される

## **認証機能**
- cloudfrontのBasic認証をLambda@edgeを利用して実装
- ログインIDとパスフレーズは予実モニターと同様
